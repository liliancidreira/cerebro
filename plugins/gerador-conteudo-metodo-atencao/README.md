# Gerador de Conteúdo — Método Atenção Digital

Geração de conteúdo da Lilian Cidreira pelo Método Atenção Digital (Elias Maman). Conhecimento embutido em `references/` (sincronizado do GitHub pelo marketplace — sem token para ler).

## Como está organizado (uma skill por formato + um orquestrador)
**Skills de formato (uso direto, dedicadas):**
- `conteudo-linkedin` — posts de texto para LinkedIn.
- `conteudo-reel` — roteiros de Reel (vídeo curto).
- `conteudo-legenda` — legendas de Instagram/TikTok.
- (carrossel é o plugin `criador-carrossel`.)

**Orquestrador (só para packs):**
- `orquestrador-conteudo` — gera um PACK (vários formatos de uma vez) para um produto. Use para "gera os conteúdos da semana", "monta o pack", "não sei o que postar".

**Skills internas (NUNCA diretas — só via orquestrador/formato):**
- `estrategista-atencao`, `pesquisador-temas`, `revisor-estrutura-reels`, `revisor-gatilhos`.

## Regra de ouro (intake)
Toda criação de conteúdo segue `references/conteudo/intake-conteudo.md`: **produto → objetivo → categoria**, e o **checklist pré-escrita é obrigatório toda vez**.
