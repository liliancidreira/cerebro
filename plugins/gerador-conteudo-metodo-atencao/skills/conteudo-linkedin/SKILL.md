# Skill: Conteúdo para LinkedIn

Escreve **posts de texto para o LinkedIn** — o formato de maior alcance orgânico para o perfil da Lilian. Cada post demonstra expertise vivida e move o leitor em direção à Lilian.

## Passo 0 — Base de conhecimento (já vem com o plugin)
O conhecimento do cérebro **já foi entregue com este plugin**, na pasta **`references/`** (sincronizada do GitHub pelo marketplace — fonte única). **Sem token, sem clonar.** Leia conforme o intake.

## Passo 1 — Intake OBRIGATÓRIO (antes de escrever)
Siga **à risca** `references/conteudo/intake-conteudo.md`:
1. **Produto** (pergunte): Signature / MPP / Método / Aceleração / MMV / Autoridade. Carregue a base do produto e a persona de `references/nucleo/produtos/...`.
2. **Objetivo** (defina): atrair / posicionar / nutrir / quebrar objeção / converter / reativar. Define o CTA e a temperatura.
3. **Categoria/ângulo** (pergunte): dor / desejo / urgência / depoimento / história pessoal / erro-mito / ensino / bastidores / opinião contrária / assunto em alta / mercado-negócios-carreira / outro.

## Passo 2 — Checklist pré-escrita OBRIGATÓRIO (toda vez)
Confirme **todos** os itens do checklist do `intake-conteudo.md` antes de escrever (persona certa, gancho forte, prova real quando o ângulo pede, gatilho do método definido, CTA coerente com o objetivo, voz/proibições, tamanho certo). Se algum item falhar, ajuste antes de escrever.

## Passo 3 — Escrever o post (estrutura de LinkedIn)
```
[ABERTURA — 1-3 linhas que param o scroll]
  Caminho A: dado real, afirmação provocadora ou pergunta direta
  Caminho B: cena concreta de uma história (do banco)

[DESENVOLVIMENTO — 3-5 parágrafos curtos de 2-4 linhas]
  Abriu com dado → traga história de cliente no meio
  Abriu com história → siga com argumentos e dados

[VIRADA — 1-2 linhas]
  A frase central que cristaliza o argumento (lugar ideal para uma analogia do banco)

[CTA — 1 linha, coerente com o OBJETIVO definido no intake]
```

**Regras (LinkedIn):**
- 200-500 palavras. Sem título editorial — começa direto na primeira linha.
- Quebras de linha generosas (1 ideia por parágrafo). Sem emojis, sem hashtags.
- Fluxo com conectivos ("porém", "então", "e um detalhe importante:") — não frases estacadas.
- CTA reflexivo costuma render mais que CTA direto no LinkedIn (a não ser que o objetivo seja converter).
- Dado de autoridade sempre dentro do fio narrativo, nunca como autodeclaração solta.
- Voz da Lilian: sem travessão, sem CAPS para ênfase, sem clichê motivacional.

Detalhes de arquétipos, gatilhos e voz: `references/skill-knowledge/metodo-atencao/` e `references/nucleo/voz-e-tom.md`. Padrões específicos de LinkedIn: `references/conteudo/referencias/linkedin-playbook.md`.

## Passo 4 — Entregar
Entregue o post + 1 linha de resumo do que foi definido (produto / objetivo / categoria / gatilho), para validação rápida.
