# Como Achar Temas com Potencial Viral — Elias Maman

> Fonte: Podcast "O Mentor" (cap. "Como encontrar temas virais no nicho"), podcast "Por Trás do Sucesso" (cap. "Conteúdo Notável") e podcast "Papo de Bastidores".

## Princípio Central

> "Existem assuntos virais que são universais que eu posso lincar com qualquer nicho." — Elias Maman

Não se trata de buscar trends ou hypes passageiros. O método é conectar **assuntos universais** (que todo mundo se identifica) com o **nicho específico** do criador.

## O Método: Assunto Universal + Nicho

### Passo 1 — Identificar Assuntos Universais

São temas com os quais todo ser humano se identifica, independente de nicho:

- **Família**: todo mundo tem mãe, pai, filhos — histórias familiares conectam.
- **Dinheiro**: rico vs pobre, renda, ganhar mais, perder dinheiro.
- **Idade**: envelhecer, juventude, fases da vida (30 anos, 40 anos, etc.).
- **Medos**: medo de fracassar, medo de ficar sozinho, medo de perder.
- **Status**: ser reconhecido, ser ignorado, ascensão social.
- **Saúde**: corpo, doença, energia, disposição.
- **Relacionamentos**: casamento, separação, solidão, atração.
- **Tecnologia/Atualidade**: IA, ChatGPT, mudanças de mercado.
- **Celebridades/Cultura pop**: Neymar, eventos midiáticos, polêmicas.

### Passo 2 — Conectar com o Nicho

Pegar o assunto universal e criar uma **ponte narrativa** para o nicho.

**Exemplo do Elias** (nicho: SaaS):
- Ruim: "Assim que você constrói um SaaS para sua empresa" → técnico, pouca gente para.
- Bom: "Eu joguei minha conta do ChatGPT no lixo depois que entendi isso aqui" → pega TODO MUNDO → depois conecta com SaaS.

**Exemplo** (nicho: equinocultura):
- Não é todo mundo que tem um cavalo, mas todo mundo sabe o que é e tem curiosidade.
- Um vídeo de curiosidade sobre cavalos atrai milhões → transição narrativa leva para o negócio.

**Exemplo** (nicho: saúde):
- "Tem um erro que você comete no seu intestino que faz acumular 5 kg de fezes" → saúde intestinal é universal.

### Passo 3 — A Transição Narrativa

Dentro do Reel, o conteúdo precisa ter uma **transição que leva do universal para o nicho**. É isso que filtra a audiência — quem se identifica com o nicho fica, quem não se identifica sai naturalmente.

## Como Gerar Ideias Rapidamente

Elias ensina que é possível criar ideias para o ano todo em menos de 30 minutos. O princípio:

1. Listar as **dores, medos, desejos e problemas** do avatar (5-10 de cada).
2. Para cada um, cruzar com **assuntos universais** da lista acima.
3. Cada cruzamento = 1 ideia de conteúdo.
4. Exemplo: Dor do avatar "não consegue delegar" + Assunto universal "dinheiro" = "Você está perdendo R$ 50 mil por ano porque não sabe delegar."

## O que Torna um Tema "Notável" (não apenas "bom")

Um tema notável precisa ter pelo menos um destes:

- **Informação nova**: dado ou perspectiva que o público não conhecia.
- **Fato curioso**: algo surpreendente que gera o "eu não sabia disso".
- **Identificação**: a pessoa se vê naquela situação.
- **Prova**: resultado concreto, número, caso real.

## Aplicação para a Mentoria Signature

### Assuntos universais a cruzar com o nicho da Lilian:
- Dinheiro / faturamento / renda
- Carreira / promoção / demissão
- Tempo / liberdade / qualidade de vida
- Status / reconhecimento / ser referência
- Medo de ficar para trás / obsolescência
- Família / equilíbrio vida pessoal e profissional

### Dores do avatar (executivos, profissionais de alta renda):
- Usar como base o brand-knowledge.md do repo cerebro.
- Cada dor cruzada com um assunto universal gera um tema.
