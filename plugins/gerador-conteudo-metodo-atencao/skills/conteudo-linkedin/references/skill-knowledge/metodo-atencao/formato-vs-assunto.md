# Formato vs Assunto — Como Escalar Conteúdos Virais — Elias Maman

> Fonte: Podcast "Por Trás do Sucesso" (cap. "Formato vs Assunto: Como Escalar Seus Conteúdos Virais") — transcrição direta.

## O Conceito

Elias separa dois elementos independentes em todo conteúdo:

- **FORMATO** = COMO você apresenta (a embalagem visual e narrativa).
- **ASSUNTO** = O QUE você fala (o tema, o conteúdo em si).

A chave para escalar: quando um formato funciona, replicar o mesmo formato com assuntos diferentes. Quando um assunto funciona, testar formatos diferentes para o mesmo assunto.

## Exemplos de Formatos

| Formato | Descrição | Gancho Visual |
|---|---|---|
| Tela dividida | Vídeo lado a lado (react, antes/depois, comparação) | Forte — dois estímulos simultâneos |
| Direto para câmera | Pessoa falando olhando para a câmera | Depende da frase de abertura |
| Objeto/Prop | Segurando ou mostrando um objeto relevante | Forte — boneco anatômico, cartão, produto |
| Lowfi / B-roll | Imagens de apoio enquanto narra por cima | Estético — funciona para storytelling |
| React | Reagindo a outro vídeo ou conteúdo | Forte — curiosidade dupla |
| Storytelling filmado | Cenas encenadas contando uma história | Cinematográfico — alto investimento |
| Lista/Ranking | "3 coisas que...", "5 erros que..." | Estruturado — expectativa de completude |
| Tutorial rápido | Passo a passo prático filmado | Prático — resolve um problema |

## Como Usar na Prática

### Escalar por formato
1. Criar um Reel que viraliza usando formato X (ex: tela dividida).
2. Repetir o mesmo formato trocando o assunto.
3. Testar 5-10 variações do mesmo formato com assuntos diferentes.
4. Manter o formato enquanto funcionar.

### Escalar por assunto
1. Criar um Reel que viraliza sobre assunto Y (ex: "delegar").
2. Repetir o mesmo assunto em formatos diferentes.
3. Testar: direto para câmera, lowfi, tela dividida, lista.
4. Identificar qual formato performa melhor para aquele assunto.

## O que o Elias Diz sobre "Lowfi"

> "Lowfi não é moda. Deram um nome para um formato que existe desde que eu produzo conteúdo."

O princípio de usar elementos visuais enquanto fala é atemporal. O nome "lowfi" é novo, mas o formato não. Isso reforça que formatos baseados em princípios (não em trends) duram mais.

## Quanto Mais Cartas na Mesa, Melhor

A frase do Elias: "Quanto mais cartas na mesa eu tiver, melhor." Isso significa:
- Combinar formato + gancho visual + assunto universal + gatilho de atenção.
- Cada "carta" é um elemento que aumenta a chance de viralizar.
- Um Reel com tela dividida + boneco anatômico + frase contraintuitiva + dado surpreendente tem 4 "cartas na mesa".

## Aplicação para a Mentoria Signature

### Formatos recomendados para testar:
1. **Direto para câmera** — Lilian falando com autoridade (marca pessoal forte).
2. **Lowfi / B-roll** — Imagens de lifestyle executivo + narração com storytelling.
3. **Tela dividida / React** — Reagindo a posts de LinkedIn, notícias de mercado, tendências.
4. **Lista** — "3 sinais de que você precisa de um mentor" / "5 erros que executivos cometem".
5. **Storytelling filmado** — Cases reais de mentorados (com autorização).

### Como escalar:
- Quando um formato funcionar, criar pelo menos 5 variações com assuntos diferentes antes de trocar.
- Manter um registro de qual formato + assunto gerou melhor resultado (usar exemplos/conteudos-aprovados.md).
