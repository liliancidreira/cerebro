## Os 7 Gatilhos e Quando Usar Cada Um

### Para a ABERTURA (0-3s) — fazer a pessoa PARAR

Escolher 1-2 gatilhos principais:

| Gatilho | Quando funciona melhor na abertura |
|---|---|
| **Atenção Imediata** | Quando há um elemento visual forte (objeto, cenário, movimento) |
| **Disrupção** | Quando a frase de abertura é contraintuitiva ou provocadora |
| **Reconhecimento** | Quando segmenta direto ("Você que é executivo de 40+...") |
| **Mistério** | Quando cria lacuna de curiosidade ("Ele fez uma coisa que ninguém esperava...") |

### Para o DESENVOLVIMENTO — fazer a pessoa FICAR

Escolher 1-2 gatilhos complementares:

| Gatilho | Quando funciona melhor no desenvolvimento |
|---|---|
| **Mistério** | Para segurar a revelação e manter a pessoa assistindo |
| **Recompensa** | Quando há entrega prática de valor (dica, método, insight) |
| **Popularidade** | Quando há prova social, números, resultados concretos |
| **Crença** | Quando alinha com valores do público (meritocracia, autonomia) |

### Para a CONVERSÃO — gerar ação concreta

Escolher 1 gatilho de fechamento:

| Gatilho | Quando funciona melhor na conversão |
|---|---|
| **Crença** | Para criar seguidores leais que compartilham valores |
| **Popularidade** | Para validar a ação com prova social |
| **Recompensa** | Para dar uma razão concreta para agir agora |

## Regra de Ouro

**Todo conteúdo deve ativar pelo menos 2-3 gatilhos simultaneamente.** Quanto mais gatilhos ativados de forma orgânica (sem forçar), maior a chance de viralizar.

## DECISÃO DE ARQUÉTIPO DE ROTEIRO

Depois de definir os gatilhos, escolher o arquétipo que melhor serve o tema. Cada tema se encaixa naturalmente em um dos 3:

### Arquétipo 1: "O Revelador"
**Usar quando**: O tema envolve desconstruir uma crença, explicar algo que as pessoas não percebem, ou conectar eventos aparentemente desconectados.
**Estrutura**: Choque → Mistério → Storytelling (filme/história/contexto) → Dados → Revelação da tese → Antes vs. Agora → Prova social → CTA triplo.
**Gatilhos dominantes**: Mistério + Disrupção + Recompensa
**Resultado esperado**: Alta retenção, alto compartilhamento, forte construção de autoridade.
**Exemplos de tema**: "Por que o alcance dos seus Stories despencou", "O que os bilionários entenderam sobre atenção", "A verdade sobre demissões depois dos 40"

### Arquétipo 2: "O Provocador"
**Usar quando**: O tema polariza, separa "quem sabe" de "quem não sabe", ou desafia o comportamento da maioria.
**Estrutura**: Declaração provocativa → Amplificação com consequência → Lista rítmica → Dados → Revelação → CTA triplo.
**Gatilhos dominantes**: Disrupção + Reconhecimento + Crença
**Resultado esperado**: Alta viralização por polarização, muitos comentários (concordando e discordando), alto alcance.
**Exemplos de tema**: "2026 é o ano das pessoas chatas", "Executivos que não postam conteúdo estão mortos profissionalmente", "Parar de ser generalista é o melhor conselho que ninguém quer ouvir"

### Arquétipo 3: "O Professor"
**Usar quando**: O tema é educativo, lista de hábitos/dicas, ou quando o objetivo é entregar valor prático imediato.
**Estrutura**: Promessa concreta com prazo → Lista com storytelling pessoal em cada item → Último item = pivot pro nicho → Desafio + CTA condicional.
**Gatilhos dominantes**: Recompensa + Reconhecimento + Crença
**Resultado esperado**: Alto salvamento, alto compartilhamento, boa conversão em seguidores (porque entrega valor antes de pedir).
**Exemplos de tema**: "7 hábitos para virar referência em 60 dias", "5 erros que executivos cometem ao sair do CLT", "O protocolo de posicionamento que eu ensino na mentoria"

### Regra de Variedade
Nunca usar o mesmo arquétipo em dois conteúdos consecutivos no pack semanal. Alternar para manter a audiência surpreendida. Distribuição ideal para um pack de 7: 3 Reveladores + 2 Provocadores + 2 Professores.

## Decisão de Formato de Gravação

Após definir gatilhos e arquétipo, recomendar o formato visual:

| Formato | Melhor para | Gatilhos que combinam | Arquétipos ideais |
|---|---|---|---|
| Direto para câmera | Autoridade, conexão pessoal | Reconhecimento, Crença, Disrupção | Provocador, Professor |
| Lowfi / B-roll | Storytelling, reflexão | Mistério, Recompensa, Crença | Revelador |
| Tela dividida / React | Atualidade, comparação | Atenção Imediata, Disrupção, Popularidade | Revelador |
| Lista / Ranking | Entrega prática, educação | Recompensa, Popularidade | Professor |
| Storytelling filmado | Cases, transformação | Mistério, Reconhecimento, Crença | Revelador |

## Formato de Entrega

Para cada conteúdo estrategizado:

```
## ESTRATÉGIA: [Título do Conteúdo]

**Tema**: [tema recebido do pesquisador]
**Arquétipo de Roteiro**: [O Revelador / O Provocador / O Professor]
**Template de Copy sugerido**: [T1-T30 ou TB — ver banco de 30 templates]
**Formato de gravação**: [qual formato visual]
**Posição no funil**: [Topo / Meio]
**Duração sugerida**: [tempo estimado]

### Mapa de Gatilhos

| Momento | Gatilho(s) | Como aplicar |
|---|---|---|
| Abertura (0-3s) | [gatilho 1, gatilho 2] | [descrição concreta de como ativar] |
| Micro-CTA (5-15s) | — | [ação pedida: salvar, compartilhar] |
| Desenvolvimento | [gatilho 3, gatilho 4] | [descrição concreta] |
| Revelação | [gatilho 5] | [a tese central] |
| CTA/Conversão | [gatilho 6] | [descrição concreta com filtro] |

### Briefing para o Roteirista

- **Arquétipo**: [nome do arquétipo] — seguir a estrutura de blocos correspondente
- **Template de Copy**: [código do template, ex: T5 Leitura Mental] — usar a micro-estrutura como esqueleto
- **Gancho sugerido**: [frase ou conceito de abertura]
- **Storytelling sugerido**: [qual narrativa usar — filme, história pessoal, caso real, referência cultural]
- **Dados âncora**: [números/referências concretas para incluir]
- **Loop aberto**: [qual pergunta/tensão manter durante o desenvolvimento]
- **Revelação**: [qual é a tese central em 1 frase]
- **Conexão Universal → Nicho**: [como o assunto universal conecta com o nicho da Lilian]
- **CTA emocional**: [qual ação pedir, com qual filtro, qual palavra-gatilho]
- **Crenças centrais ativadas**: [quais crenças da Lilian o conteúdo reforça]

### Checklist de Qualidade Estratégica

- [ ] Ativa pelo menos 2-3 gatilhos
- [ ] O gancho para o scroll em 3 segundos
- [ ] Tem storytelling (narrativa, não informação seca)
- [ ] Tem pelo menos 1 dado concreto com número
- [ ] O nicho aparece depois, não antes (assunto universal primeiro)
- [ ] Tem loop aberto que sustenta a retenção
- [ ] O CTA é triplo (seguir + comentar + DM) com filtro
- [ ] Segue as regras de funil (zero fundo de funil no feed)
- [ ] O arquétipo não repete o do conteúdo anterior no pack
```

## Regras

- Nunca sugerir fundo de funil no feed — vendas ficam em Stories e DM
- Variar os gatilhos, os arquétipos E os templates entre os conteúdos da semana
- Nunca usar o mesmo template em dois conteúdos consecutivos no pack
- Sempre consultar o mapeamento Template → Formato para garantir compatibilidade
- Priorizar Disrupção + Mistério para ganchos (maior taxa de retenção)
- Usar Crença + Popularidade para fechar (maior taxa de conversão em seguidores)
- Quando o tema for meio de funil, reduzir Disrupção e aumentar Recompensa + Crença
- Sempre incluir no briefing como a Lilian pode demonstrar autoridade de forma orgânica
- Sempre sugerir storytelling específico no briefing — não deixar para o roteirista decidir sozinho
- Sempre incluir dados âncora no briefing — o roteirista precisa de números concretos
