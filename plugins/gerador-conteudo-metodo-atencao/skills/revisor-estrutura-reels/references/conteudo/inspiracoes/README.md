# Biblioteca de Inspirações — fonte transversal de ângulos e ganchos

Curadoria de **livros e ideias que a Lilian se interessou**, traduzidos para a voz dela e para o nicho. Não é "frase motivacional": é um banco de **princípios + ganchos + pontes pro nicho** que dá lastro autoral ao conteúdo.

## Como as skills usam (transversal, não por categoria)
Esta biblioteca é uma **fonte de ideias como o banco de histórias e as personas** — consultada de forma **inteligente em QUALQUER peça**, durante a busca do ângulo/gancho. A skill puxa um princípio daqui quando ele **fortalece** o conteúdo do produto + objetivo + categoria escolhidos, sempre:
- fazendo a **ponte pro nicho** (carreira / posicionamento / virar mentor), nunca a citação solta;
- respeitando a `voz-e-tom` — **sem clichê motivacional** ("acredite no seu potencial"). Aqui é reflexão **com lastro**: princípio real + a bagagem vivida da Lilian + um ângulo concreto.

Existe também a categoria "Reflexivo / motivacional (com lastro)" no intake para quando a Lilian quiser, de propósito, uma peça reflexiva — mas a biblioteca **não fica presa a ela**; alimenta todas as categorias.

## O que tem aqui
- `napoleon-hill.md` — princípios e ganchos a partir dos grifos da Lilian em Napoleon Hill.
- `_template.md` — modelo para adicionar um livro novo.
- *(novos livros entram como um arquivo cada, conforme a Lilian for lendo.)*

## Como adicionar um livro novo
A Lilian manda o livro (ou os grifos dela) → o conteúdo é **curado** no formato do `_template.md` (princípios traduzidos para a voz + ganchos + pontes pro nicho) → commit no cérebro → o auto-sync entrega para todas as skills. A curadoria pede síntese, então é feita com o Claude (ou pela Lilian, guiada pelo template).
