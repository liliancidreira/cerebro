# Núcleo — conhecimento que serve para TUDO

Este é o **núcleo compartilhado** do cérebro da Lilian Cidreira. Tudo aqui é transversal: serve para **criar conteúdo, vender, agendar sessão, prospectar** — qualquer skill.

## Regra de governança (inviolável)
- O GitHub é a **única fonte de verdade**. Nenhuma skill usa informação interna/embutida.
- Toda skill, no **Passo 0**, sincroniza o repositório `cerebro` e lê **`/nucleo`** (este diretório) + o seu cérebro específico (`/conteudo` ou `/vendas`).
- Atualizou aqui uma vez → conteúdo e vendas já leem a versão nova. **Sem duplicação.**

## O que tem aqui
- `produtos/` — perfil de cada produto (o que é, ICP, dores, ambições, estopim, sinais de perfil, como a pessoa fala, promessa). **Um arquivo por produto.**
- `ofertas-e-precos.md` — **fonte única de preços e condições** de todos os produtos. Os arquivos de produto não repetem preço; apontam para cá.
- `provas-e-depoimentos.md` — **fonte única do banco de casos, números e depoimentos**. Os arquivos de produto referenciam casos pelo nome; o banco completo está aqui.
- `banco-de-historias.md` — **histórias reais da Lilian e de mentorados** (narrativas), transversais a vendas e conteúdo. Resultados numéricos ficam em `provas-e-depoimentos.md`; aqui ficam as narrativas.
- `fatos.md` — fatos da Lilian e da Future Minds (datas, números institucionais).
- `voz-e-tom.md` — voz, tom e proibições de linguagem.

## O que NÃO fica aqui (vai para os cérebros específicos)
- Como **qualificar e rotear** um lead (origem do lead, matriz desejo × R$11k, MPP × Método, condução da Masterclass, "como a sessão vende"), SPIN, objeções, prospecção → **`/vendas`**.
- Método Atenção Digital, gatilhos, arquétipos de roteiro, templates de carrossel, banco de ganchos → **`/conteudo`**.
