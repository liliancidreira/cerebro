# Arquétipos de roteiro + DNA do roteiro

## O DNA do roteiro (presente em TODO roteiro, qualquer formato)
1. **A primeira frase É o gancho** — nunca saudação/contexto/"Hoje vou falar sobre…". A pessoa decide em 3s.
2. **Micro-CTA nos primeiros 15s** — pedir uma ação pequena ("salve", "compartilhe") antes do conteúdo principal.
3. **Storytelling como veículo** — nunca informação seca. Cada ponto envolto em narrativa (história pessoal, referência cultural, caso real com nome e números).
4. **Dados concretos como âncora** — sempre pelo menos um número específico. Generalização mata a credibilidade.
5. **O nicho aparece DEPOIS, não antes** — o assunto universal vem primeiro; a conexão com carreira/posicionamento surge organicamente no desenvolvimento ou no CTA.
6. **CTA triplo padronizado** — 3 camadas em sequência: follow + comentário (palavra-gatilho) + promessa de retorno (DM/link/próximo passo).
7. **Filtro no CTA** — nunca para "todo mundo"; um filtro que seleciona e compromete ("compartilhe com quem tem os olhos abertos", "só se você topar o desafio").

## Os 3 arquétipos

### 1. O Revelador
- **Quando usar:** desconstruir uma crença, explicar algo que ninguém percebe, conectar eventos aparentemente desconexos.
- **Estrutura:** Choque → Mistério → Storytelling (filme/história/contexto) → Dados → Revelação da tese → Antes vs. Agora → Prova social → CTA triplo.
- **Gatilhos dominantes:** Mistério + Disrupção + Recompensa.
- **Duração ideal:** 2-3 min. **Funil:** topo (atrai audiência massiva).
- **Resultado:** alta retenção, alto compartilhamento, forte autoridade.

### 2. O Provocador
- **Quando usar:** temas que polarizam, separam "quem sabe" de "quem não sabe", desafiam o comportamento da maioria.
- **Estrutura:** Declaração provocativa → Amplificação com consequência → Lista rítmica → Dados → Revelação → CTA triplo.
- **Gatilhos dominantes:** Disrupção + Reconhecimento + Crença.
- **Resultado:** alta viralização por polarização, muitos comentários, alto alcance.

### 3. O Professor
- **Quando usar:** educativo, lista de hábitos/dicas, entregar valor prático imediato.
- **Estrutura:** Promessa concreta com prazo → Lista com storytelling pessoal em cada item → Último item = pivot pro nicho → Desafio + CTA condicional.
- **Gatilhos dominantes:** Recompensa + Reconhecimento + Crença.
- **Resultado:** alto salvamento, alto compartilhamento, boa conversão em seguidores.

## Regra de variedade
Nunca usar o mesmo arquétipo em dois conteúdos consecutivos. Distribuição ideal para um pack de 7: **3 Reveladores + 2 Provocadores + 2 Professores**.
