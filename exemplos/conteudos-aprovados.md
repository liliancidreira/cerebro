# Conteúdos Aprovados — Registro de Performance

> Registro dos conteúdos que performaram bem para análise de padrões e replicação.

## Como Usar

Após cada quinzena de conteúdo, registrar aqui os Reels que tiveram melhor performance.
Isso permite identificar quais combinações de formato + assunto + gatilho funcionam melhor para o perfil da Lilian.

## Template de Registro

### [Data] — Título do Conteúdo
- **Formato**: (direto para câmera / lowfi / tela dividida / lista / storytelling)
- **Assunto Universal**: (dinheiro / carreira / tempo / status / medo / família / tecnologia)
- **Dor do Avatar**: (qual dor específica o conteúdo endereçou)
- **Gatilhos Usados**: (quais dos 7 gatilhos foram ativados)
- **Funil**: (topo / meio)
- **Métricas**:
  - Taxa de pulagem: ___%
  - Tempo médio de retenção: ___% (___s de ___s total)
  - Taxa de interação: ___%
  - Alcance: ___
  - Novos seguidores: ___
- **O que funcionou**: (análise qualitativa)
- **O que melhorar**: (ajustes para próxima vez)
