# Os 7 gatilhos de atenção (Método Atenção Digital)

Regra de ouro: **todo conteúdo deve ativar pelo menos 2-3 gatilhos simultaneamente**, de forma orgânica. Quanto mais gatilhos ativados sem forçar, maior a chance de viralizar.

## Para a ABERTURA (0-3s) — fazer a pessoa PARAR (escolher 1-2)
- **Atenção Imediata** — quando há um elemento visual forte (objeto, cenário, movimento).
- **Disrupção** — quando a frase de abertura é contraintuitiva ou provocadora.
- **Reconhecimento** — quando segmenta direto ("Você que é executivo de 40+…").
- **Mistério** — quando cria lacuna de curiosidade ("Ele fez uma coisa que ninguém esperava…").

## Para o DESENVOLVIMENTO — fazer a pessoa FICAR (escolher 1-2)
- **Mistério** — para segurar a revelação e manter assistindo.
- **Recompensa** — quando há entrega prática de valor (dica, método, insight).
- **Popularidade** — quando há prova social, números, resultados concretos.
- **Crença** — quando alinha com valores do público (meritocracia, autonomia).

## Para a CONVERSÃO — gerar ação (escolher 1)
- **Crença** — para criar seguidores leais que compartilham valores.
- **Popularidade** — para validar a ação com prova social.
- **Recompensa** — para dar uma razão concreta de agir agora.

## Heurísticas
- Ganchos: priorizar **Disrupção + Mistério** (maior retenção).
- Fechamento: **Crença + Popularidade** (maior conversão em seguidores).
- Meio de funil: reduzir Disrupção, aumentar **Recompensa + Crença**.
