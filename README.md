# Cérebro — Lilian Cidreira

Este repositório é a memória persistente da skill de geração de conteúdo. A cada sessão, a skill lê estes arquivos antes de gerar qualquer post, reel ou carrossel. A cada história coletada ou ajuste aprovado, estes arquivos são atualizados aqui.

## Arquivos

- **banco-de-historias.md** — Histórias reais da trajetória da Lilian, de alunos e padrões observados em campo. Fonte primária de storytelling — nunca inventar, sempre usar o que está aqui.
- **voz-e-aprendizados.md** — Regras de voz identificadas através dos ajustes que Lilian faz nos posts gerados. Cada correção vira uma regra registrada aqui.
- **skill-knowledge/metodo-atencao/** — Base de conhecimento extraída do Método Atenção Digital de Elias Maman (7 gatilhos, 5 pilares, roteiro explosivo, temas virais, formato vs assunto, métricas, funil).
- **exemplos/conteudos-aprovados.md** — Registro de conteúdos publicados com métricas de performance para análise de padrões.
