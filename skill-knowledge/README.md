# Skill Knowledge — Conhecimento Base da Skill

Esta pasta contém os arquivos de referência que formam o conhecimento base da skill de geração de conteúdo.
São a fonte de verdade — se a skill for reinstalada do zero, estes arquivos são usados para repovoá-la.

## Arquivos

**`brand-knowledge.md`** — Tudo sobre a Lilian, o produto (Mentoria Signature), o público-alvo, a voz, o que é proibido e o vocabulário de marca. Inclui todos os aprendizados de sessões anteriores.

**`content-templates.md`** — Estrutura exata de cada tipo de conteúdo (Reel, Carrossel, Post de Texto), com todas as regras de qualidade, CTA, fluxo de parágrafo e princípios de viralidade.

**`storytelling-analogias.md`** — Os 4 arquétipos de história e o banco de 9 analogias calibradas para o perfil do cliente ideal (executivos de alta renda).

## Quando atualizar

Estes arquivos são atualizados sempre que:
- Lilian aprova um ajuste de voz que gera uma nova regra
- Um novo vocabulário de marca é identificado
- Uma nova analogia é desenvolvida
- Um template de conteúdo é refinado
